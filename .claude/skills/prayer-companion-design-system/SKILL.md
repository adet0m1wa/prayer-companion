# Prayer Companion Design System

This skill contains the complete design system for the Prayer Companion app — a Bible-focused app with a calm, intentional manuscript aesthetic.

## Token Files
- `tokens/colors.json` — 9 color tokens across 3 groups (ink, accent, surface)
- `tokens/typography.json` — 11 text styles across 3 font families
- `tokens/spacing.json` — 10 spacing values, base unit 2px
- `tokens/icons.json` — 5 icon sizes
- `tokens/radius.json` — 5 corner radius values

## Component Patterns
- `components/patterns.md` — reusable component structures across all screens

## Rules
