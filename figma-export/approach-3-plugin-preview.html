APPROACH 3: FIGMA PLUGIN API

Screen 1 — Daily View v3

This approach uses the Figma Plugin API to programmatically create every frame, text node, and shape. The full script is saved at:
figma-export/approach-3-figma-plugin-script.js

How to use: Open Figma → Plugins → Development → New Plugin → paste the script from approach-3-figma-plugin-script.js → Run. It creates the full Screen 1 v3 layout with auto-layout frames, proper typography, and color tokens.

// Key structure created by the plugin:

const screen = figma.createFrame();
screen.name = "screen-1-daily-view-v3";
screen.resize(402, 874);
screen.fills = [{ color: "#E4DED5" }];
screen.layoutMode = "VERTICAL";

// Status Bar (52px)

Limitations: The Figma REST API is read-only — it cannot create or modify design nodes. The Plugin API (shown here) is the only way to programmatically build Figma designs. Icons use placeholder shapes since Phosphor font isn't available in plugins. Images (parchment textures, waterfall) require manual import after the plugin runs.
